# 全栈开发工具与笔记网站

## 项目简介

这是一个专注于全栈开发的文档型网站，整理了各种开发工具的应用指南、常用链接和PDF书籍资源，旨在为开发者提供一个实用的工具性网站。

## 项目结构

```
my_note/
├── index.md              # 常用工具应用指南（Eureka等）
├── fav-links.md          # 常用链接工具
├── pdfs/                 # PDF书籍资源
│   ├── index.md          # PDF书籍列表
│   ├── JavaScript权威指南(第6版).pdf
│   ├── 重构：改善既有代码的设计(第2版).pdf
│   └── 高性能mysql+第四版+英文版.pdf
└── README.md             # 项目说明文档
```

## 主要内容

### 1. 常用工具应用指南

- **Eureka**：服务注册与发现的详细指南
- **Java/Spring Boot常用注解**：各种注解的区别与联系
- **MyBatis缓存**：深入解析MyBatis的缓存机制
- **JPA与MyBatis对比**：两种ORM框架的优缺点分析
- **ORM框架**：对象关系映射的基本概念与应用

### 2. 常用链接工具

- **计算机各大工具网址**：Docker、Redis、Java、MySQL、RabbitMQ、MDN等官方网站
- **编程工具官方**：VSCode、JetBrains Idea等开发工具官网
- **在线实用工具**：Excalidraw（在线手绘）、I Love PDF（文件格式转换）、UTools（效率工具）

### 3. PDF书籍资源

提供了多本技术书籍的PDF版本，包括：
- JavaScript权威指南(第6版)
- 重构：改善既有代码的设计(第2版)
- 高性能mysql+第四版+英文版

## 如何使用

### 访问网站

直接通过以下链接访问网站：
- [首页](https://wujingkai-nsm.github.io/)
- [PDF书籍](https://wujingkai-nsm.github.io/pdfs)
- [常用链接工具](https://wujingkai-nsm.github.io/fav-links)

### 本地运行

1. 克隆仓库到本地
   ```bash
   git clone https://github.com/wujingkai-nsm/wujingkai-nsm.github.io.git
   ```

2. 使用任意静态网站服务器运行
   ```bash
   # 使用Python启动简单服务器
   python -m http.server 8080
   ```

3. 在浏览器中访问 http://localhost:8080
