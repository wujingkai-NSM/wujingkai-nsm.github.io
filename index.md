# Wujingkai

## 专业技能
- 熟悉计算机网络 **TCP/UDP**、**SMTP**、**HTTP** 等协议，熟悉 OSI 网络七层模型，掌握 WebSocket/SSE 技术。
- 数据结构，树、图、链表，掌握 Redis 底层存储的跳表（Skip List）结构、MySQL 索引底层存储的 B+树结构；
- Java常用的数据结构：Deque、Stack 等，知晓其 Java 实现数据结构的常见的操作，和对应的渐进时间复杂度。
- 掌握集合框架（Collection、Map）、反射、注解、缓冲区、异常处理、泛型、I/O 流、流式编程。
- 深入理解 OOP 特性，熟悉 JVM 内存模型、垃圾回收机制；深入理解同步、显式锁(Explicit Lock)，原子变量等。
- 熟悉 Java I/O 编程，掌握 **IO/NIO** 技术 如 Channel/Selector 等。**掌握并发编程**，线程池管理、锁机制ReentrantLock/Synchronized/Semaphore/CountDownLatch/CyclicBarrier。
- 掌握 Spring IOC、AOP、事务管理、拦截器/过滤器等；
- 掌握 Spring Could Alibaba 全家桶，Nacos、Ribbon、Sentinel、Gateway、Feign 组件的基本使用。
- 熟悉 MySQL、SQL Lite、Oracle、Redis；掌握事务、游标、索引、MVCC、视图等概念
- 熟悉 Postman、Apifox 等接口调试工具；熟悉 CI/CD 流程；熟悉Docker容器技术；熟悉 Jmeter 等测试工具。
- 熟悉前端框架并知晓其异同点；如 React/Vue 框架，熟悉 **JavaScript/TypeScript** 编程语言，掌握其语言特性，如闭包，Promise 异步编程等；熟悉 **Vite/Webpack** 打包工具。
- 掌握 Linux 的文件系统/用户管理/组管理/系统监控；熟悉整个软件的部署流程；掌握Git/SSH/Vim/Shell 脚本等服务端工具的基本使用。
- 了解 Nginx相关配置 、负载均衡，高并发请求分发。
- 了解密码学、信息安全等相关概念，掌握常用的****加密算法，AES、RSA****；哈希算法 MD5、SHA。
- 掌握在业务中对对称加密和非对称加密算法的基本使用；了解重放攻击，中间人等安全问题。
